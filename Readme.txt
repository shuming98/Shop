本网站使用非常简单.
该网站目前功能：
前台：商品展示、商品详细信息浏览，加入购物车、支付、
后台：增删改查分类和商品
如果你点了某链接没反应，那就是改功能并没有写，请不要展示给别人看

1.首先，百度 WAMP/XAMPP/Ampps 集成开发环境，下载安装后，开启服务
2.把Shop文件扔进 www 网站根目录下
3.你可以给网站配置一个本地域名，这涉及到apache（httpd）和host文件配置,我相信上课老师有讲，不懂可百度一下
4.打开浏览器，输入：域名(127.0.0.1/路径)/index.php/Admin/Cat/Catelist 可添加分类[先添加一级,再二级]
5.浏览器，输入：域名(127.0.0.1/路径)/index.php/Admin/Goods/goodslist 可添加商品[到二级分类]
6.你可以随便修改 /View 目录下的前端内容，如果你没有学过PHP和对MVC模式有所了解，不建议你修改 /Model 和 /Controller 目录下的后端代码
7./Shop/Shop/Common/Conf.php 是数据库配置文件，请修改‘DB_PWD'内容为你的数据库密码（123456）
